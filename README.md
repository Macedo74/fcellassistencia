# 📱 FCell Assistência Técnica

Este é o site institucional da **FCell Assistência**, desenvolvido para apresentar serviços de manutenção de smartphones e venda de acessórios de forma profissional e responsiva.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as melhores práticas aprendidas no **DevClub**, focando em performance e design moderno:

* **HTML5**: Estruturação semântica do conteúdo.
* **Tailwind CSS**: Estilização via classes utilitárias para um design rápido e responsivo.
* **JavaScript**: Implementação de funcionalidades interativas (como o botão de voltar ao topo).
* **Font Awesome**: Biblioteca de ícones profissionais.
* **GitHub Actions**: Deploy automatizado para o GitHub Pages.

## ✨ Funcionalidades

- [x] **Design Responsivo**: Adaptado para celulares, tablets e computadores.
- [x] **Seção de Serviços**: 6 cards detalhando os principais serviços (Troca de tela, bateria, placa, etc).
- [x] **Botão Voltar ao Topo**: Funcionalidade dinâmica com JavaScript para melhorar a navegação.
- [x] **Botão de Orçamento**: Link estratégico para conversão de clientes.
- [x] **Menu Fixo (Sticky)**: Facilita o acesso à navegação em qualquer parte da página.
